Eight checks. One command.

Everything a reviewer would catch, automated.

🔍

Citation Verification

Verifies every bib entry against CrossRef. DOIs, titles, authors, journals. Catches fabricated references.

⚠️

Overclaim Detection

Flags "proven," "validated," "novel," "impossible." Suggests precise alternatives reviewers won't challenge.

🔢

Number Consistency

Cross-checks every number across abstract, body, tables, and captions. Flags contradictions.

⚙️

Parameter Auditing

Verifies code parameters match paper claims. Catches docstring drift and steady-state errors.

📈

Statistical Auditing

Checks p-values, sample sizes, test appropriateness. Catches non-significance misread as equivalence.

🔗

Evidence Mapping

Traces every claim to supporting code or data. If a claim has no evidence trail, it gets flagged.

▶️

Reproducibility

Runs referenced scripts, captures output, compares to paper numbers. Flags discrepancies >1%.

👁

Adversarial Review

Compiles all findings into a simulated hostile peer review with the most damaging objections.
